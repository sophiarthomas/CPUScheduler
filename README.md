# CPUScheduler
## Compile and Run Program for each Scheduling Algorithm 

### First-Come First-Serve
make fcfs <br />
./fcfs schedule.txt

### Round Robin
make rr <br />
./rr rr-schedule.txt

### Priority 
make priority <br />
./priority pri-schedule.txt


Note: you can replace .txt files with any file with the format
- [name], [priority],[burst]
